**Trading Features:**
- **Order Types**: Market, Limit, Stop, Stop-Limit
- **Matching**: Price-time priority (FIFO)
- **Latency**: Sub-millisecond matching
- **Throughput**: 100K+ orders per second

### 2. Wallet Infrastructure

Secure cryptocurrency storage with hot and cold wallet segregation.

```mermaid
sequenceDiagram
    participant User
    participant API as Wallet API
    participant Hot as Hot Wallet
    participant Cold as Cold Storage
    participant Blockchain as Blockchain Network

    User->>API: Deposit Request
    API->>Blockchain: Generate Address
    Blockchain->>API: Deposit Address
    API->>User: Display Address

    Note over Blockchain: User sends crypto

    Blockchain->>Hot: Incoming Transaction
    Hot->>Hot: Confirm (6 blocks BTC)
    Hot->>API: Credit User Balance

    Note over Hot,Cold: Periodic sweep to cold

    Hot->>Cold: Sweep Excess Funds
    Cold->>Cold: HSM Signing (Multi-sig)
```

**Wallet Architecture:**
- **Hot Wallets**: Online, for immediate withdrawals (~2% of assets)
- **Cold Storage**: Offline, HSM-secured (~98% of assets)
- **Multi-signature**: Requires multiple approvals
- **Geographic Distribution**: Keys spread across locations
